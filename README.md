# calculator-project

Started off as the first Mavericks Onboarding project

## Training

- Solid principles
- HTML
- CSS
- Javascript

## Initialising the Calculator Project

First step, you have to download all the modules present in the file

Run:
```npm install``` if you already have node.js, else you can download node.js on their website

Second step, to access the website, you can try opening the html file itself or utilise live server to open the html file (recommended)

## Running unit tests

Unit tests in the project is coded using Jest. In order to run the tests, 

Run:
```npm test```

There are 26 individual unit tests for the functions, they are inside the .test.js files
